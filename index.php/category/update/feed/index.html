I’m creating an information portal for anything and everything surrounding ERC20/ERC918-compliant Mineable Tokens on Ethereum. Of course there is a lot of 0xBTC info on this page because it was the first and is the largest mineable token on Ethereum. A handful of others have sprung up over the past months, many not worth even mentioning, but as Token mining begins to gain traction in the innovative Ethereum circles you will be seeing a lot more DApps and smart contracts utilizing the EIP: 918 Mineable Token Standard. This is a first draft I’m just trying to get out there and you can except a lot more updated info on new mineable tokens projects. There’s a lot of good information surrounding mineable tokens but I wanted to create a information portal type of website to put it all in one place. A place we can learn from and point others for education. I’m open to any and all criticism and ideas and help so stay tuned for for a lot more updates quite often!
